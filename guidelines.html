EU Project Guidelines

Key resources for Erasmus+ project management

Erasmus+ Programme Guide

The official Erasmus+ Programme Guide is the primary reference for all project types. It covers eligibility, application procedures, and implementation rules.

Lump-Sum Funding Model

Since 2022, many Erasmus+ projects use the lump-sum funding model. Key principles:

  • Budget is divided into work packages, not cost categories
  • Payment is tied to WP completion, not actual costs
  • No financial reporting of individual expenses required
  • WP1 (Project Management) should not exceed 20% of total budget
  • All WP deliverables must be completed for payment release

Lump-Sum Amounts (2024-2027)

  • KA220: €120,000 / €250,000 / €400,000
  • KA210: €30,000 / €60,000
  • KA15X: Variable amounts based on activity type

Reporting Requirements

Projects must submit a final report within 60 days of the project end date. The report should include:

  • Summary of activities and results per work package
  • Description of project outcomes and impact
  • Dissemination and sustainability plan
  • Financial summary (for lump-sum: WP completion declaration)
