Platform Overview

EUProject Hub is organized around these main sections:

  • Dashboard — Overview of all your projects with aggregated statistics
  • Overview — Detailed view of a selected project with Gantt chart, timeline, and partner map
  • Partners — Manage partner organizations with dual contacts and invitations
  • Work Packages — Create WPs with activities (A1, A2...), track progress and completion
  • Tasks — Kanban board, list, and calendar views with filters
  • Documents — File management with folders, drag & drop upload
  • Budget & Finance — Lump-sum budget tracking, partner transfers, compliance alerts
  • Dissemination — Log events, publications, social media reach
  • Meetings & TPMs — Plan transnational meetings with agendas
  • AI Report Generator — AI-powered final report creation
  • Settings — Profile, subscription, language, dark mode

Supported Project Types

  • Erasmus+ KA220-HED/VET/SCH/ADU/YOU (Cooperation Partnerships)
  • Erasmus+ KA210-HED/VET/SCH/ADU/YOU (Small-Scale Partnerships)
  • Erasmus+ KA152/153/154 (Youth Projects)
  • Erasmus+ KA171 (International Credit Mobility)
  • Erasmus+ Sport
  • Other EU Programmes

Keyboard Shortcuts

Dark mode toggle is available in the top bar. Language can be changed in Settings.

Data Export

Export your project data as CSV (spreadsheet-compatible) or JSON (full backup). AI reports can be exported as PDF.
